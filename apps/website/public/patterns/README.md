# TSC Patterns Folder

This folder contains all decorative patterns, textures, logos, and SVG assets used throughout the TSC website.

## Structure

- **logos/** - Brand logos and wordmarks
- **textures/** - Pattern textures and background elements
- **decorative/** - SVG decorative elements for sections
- **overlays/** - Pattern overlays for semi-transparency effects

## Usage

Import pattern assets throughout the website:

```typescript
// Example usage in components
<img src="/patterns/logos/tsc-logo.svg" alt="TSC Logo" />
<div style={{ backgroundImage: 'url(/patterns/textures/dotted.png)' }} />
```

## Recommended Patterns

- Dotted patterns for section backgrounds
- Geometric patterns inspired by Indian design
- TSC brand color variations for logos
- Decorative SVG elements for visual hierarchy
